# PercentageCalculator

Take an array of integer values and return the percentage that sums up to 100

[![Deploy to Azure](http://azuredeploy.net/deploybutton.png)](https://azuredeploy.net/)

```json
{
  "data": [
  {
    "text": "red",
    "value": 30
  },
  {
    "text": "green",
    "value": 20
  },
  {
    "text": "blue",
    "value": 150
  },
  ]
}
```
